# Freight-Cost-Analysis-Project
A data analysis project that explores freight shipment costs and delivery performance across multiple carriers and transport modes. The project highlights cost variances, operational inefficiencies, and actionable insights to support smarter logistics and supply chain decision-making.

This project analyzes freight shipment data to understand cost behavior, delivery performance, and operational efficiency across different routes, carriers, and transportation modes.
